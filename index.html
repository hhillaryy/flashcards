Flashcards

  • Javascript

    • JavaScript is the programming language that runs in the browser, which is used to build advanced user interactive Web sites and applications for secure browser execution

  • Modulo

    • Symbol that looks like %. Finds remainder between values.

  • Operators

    • Arithmetic operators are used to perform arithmetic between variables and/or values.

  • NaN

    • A value that means Not A Number.

  • Variable naming conventions

    • A set of rules for naming a variable. For example: camel case (myFriend)

  • +=

    • A shortcut to revalue and add to a named variable. For example:If myNumber=6 then myNumber += 1; will add and revalue it to 7.

  • Function

    • A JavaScript function is a block of code designed to perform a particular task.

  • Method

  • Argument to a method

  • String

    • A JavaScript string simply stores a series of characters like "John Doe". A string can be any text inside quotes.
